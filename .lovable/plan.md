
## The Pink Rosette — Full Website Redesign

**Brand:** The Pink Rosette — A premium Mumbai-based patisserie & café (Est. 2018)
**Palette:** Blush Royale — Soft pink `#F9E4E8`, Rose `#D4A5B0`, Mauve `#B56576`, Deep Burgundy `#6B2D3E`, Cream `#FDF6F0`
**Fonts:** Playfair Display (headings) + Lato (body)
**Vibe:** Luxury patisserie — romantic, refined, animated, Awwwards-level elegance

---

### Pages & Sections

#### 1. 🏠 Home Page (`/`)
- **Announcement bar** — "All orders delivered within 24–48 hrs, 12pm–6pm" with scrolling marquee animation
- **Sticky Navbar** — Logo center, nav links left/right, cart + social icons. Smooth fade-in on scroll
- **Hero Section** — Full-screen image carousel (4 slides) with elegant Ken Burns zoom animation, overlay text "Crafted with Love. Since 2018." + CTA button
- **TPR Signatures** — Horizontal scroll card grid with hover lift effect, product name, price tag in burgundy, "Choose Options" CTA
- **Featured Collections Banner** — Split-screen layout: Fleur Delice (left) vs TPR Bespoke (right), fade-in-on-scroll
- **About Snippet** — Centered quote section with decorative rose divider animation
- **Instagram Feed Teaser** — 6-image grid with hover overlay effect showing "@thepinkrosette"
- **Footer** — Logo, navigation, social links, copyright

#### 2. 🌸 Fleur Delice Collection (`/fleur-delice`)
- **Page Hero** — Full-width banner with "Fleur Delice Collection" in Playfair Display, soft pink gradient overlay
- **Product Grid** — 3-column masonry-style grid, cards with image hover zoom, product name, price in mauve, "Choose Options" button
- Products: Flowers in Bloom (₹6,500), Enchanted Garden (₹6,000), Blush Cascade, Rosette Dream, and more

#### 3. 🎂 TPR Bespoke Collection (`/tpr-bespoke`)
- **Page Hero** — Darker burgundy overlay with elegant serif heading
- **Subcategory Tabs** — Filter tabs: All, Birthday, Wedding, Celebration, Custom
- **Product Grid** — Same animated card grid as Fleur Delice but with a warmer palette
- Products: Signature Chocolate Cake (from ₹1,500), Victorian Vanilla Cake, Caramel Cake, Chocolate Hazelnut

#### 4. 🎨 Customise With Us (`/customise`)
- **Split Hero** — Left: beautiful bespoke cake image; Right: heading "Make it Yours" + description

- **Enquiry Form** — Name, Email, Phone, Event Date, Cake Description textarea, "Send Enquiry" button with hover animation

#### 5. 🍰 Gourmet Curations (`/gourmet`)
- **Page Hero** — Warm cream background with elegant heading
- **Category Cards** — Grid of gourmet product categories with fade-in scroll reveal
- **Product Listings** — Same card grid layout, styled in warm tones

#### 6. 📍 Our Cafes (`/our-cafes`)
- **Hero Banner** — Image of the Fort café with overlay text
- **Café Card: Fort, Mumbai** — Image + description "Our flagship Patisserie & Café nestled in the heart of Fort, Mumbai" + "Get Directions" button (links to Google Maps)

#### 7. 📞 Get In Touch (`/get-in-touch`)

### Global Animations
- **Scroll reveal** — Elements fade + slide up as user scrolls
- **Navbar** — Transparent → solid background on scroll
- **Cards** — Hover: lift shadow + slight scale on image
- **Buttons** — Color fill sweep animation on hover
- **Page transitions** — Smooth fade between routes
- **Hero carousel** — Ken Burns zoom + crossfade slides
